## Dependencies

- Opencv 4.5.1
- CMake 3.27

## Usage

### Linux

```sh
./neuro-eye
```

### Windows

```sh
./Release/neuro-eye.exe
```

## Build

1. ビルド用ディレクトリの作成

```sh
mkdir build
cd build
```

2. cmake  

```sh
cmake ..
```

3. ビルド

```sh
cmake --build .
```

(Windows用リリースビルドコマンド)  

```sh
cmake --build . --config Release
```
